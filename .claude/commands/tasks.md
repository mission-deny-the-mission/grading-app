Generate actionable task lists from implementation plans.

This command breaks down technical plans into specific, actionable tasks with dependencies, time estimates, and acceptance criteria.

Usage:
/tasks [options]

Options:
--detailed - Include detailed subtasks and dependencies
--priority - Order tasks by priority and dependencies
--estimate - Include time estimates for each task

Examples:
/tasks - Generate basic task list
/tasks --detailed - Create comprehensive task breakdown
/tasks --priority - Get prioritized task order
